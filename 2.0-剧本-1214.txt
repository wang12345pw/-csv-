## 剧本格式规范

### 格式符号

#### 1. 场景标记
```
※ 场景名 - 时间
```

**示例**：
```
※ 王府大殿 - 日
※ 镇国将军府后花园 - 晨
※ 书房 - 夜
```

#### 2. 旁白格式
```
【旁白】旁白内容
```

**使用场景**：
- 环境描述
- 动作描写
- 时间过渡
- 心理外化

**示例**：
```
【旁白】王府大殿内，宾客满堂，一片喜庆。
【旁白】苏映雪沉默着摘下凤钗，平静走出王府。
【旁白】三天后——
```

#### 3. 对话格式
```
角色名："对话内容"
```

**规则**：
- 角色名后用中文冒号
- 对话内容用中文引号
- 一行一句对话

**示例**：
```
顾棠之："娇娇怀了本王的骨肉，从今日起就住主院吧。"
苏映雪："是吗？那就走着瞧。"
```

### 内容规范

#### 字数要求

| 项目 | 要求 |
|------|------|
| 单集总字数 | 3000-4000字 |
| 场景数量 | 8-15个/集 |

#### 旁白与对话比例（重要！）

| 内容类型 | 占比 | 说明 |
|----------|------|------|
| 旁白【旁白】 | **≥70%** | 环境描述、动作描写、时间过渡、心理外化 |
| 对话 | **≤30%** | 角色台词，精简有力 |

**⚠️ 强制规则**：
- 旁白为主，对话为辅
- 每个场景以旁白描述为主体
- 用旁白推动剧情，用对话点睛

**⚠️ 对话角色限制**：
- **只有主角和多次出现的重要配角才能有对话**
- 路人、临时角色、只出场一次的角色**禁止有对话**
- 非核心角色的信息用旁白描述，不要硬塞对话

**对话角色判断标准**：
| 角色类型 | 是否可以有对话 | 示例 |
|---------|---------------|------|
| 主角 | ✅ 可以 | 女主、男主 |
| 重要配角（出场3次+） | ✅ 可以 | 男主母亲、女主闺蜜 |
| 次要配角（出场1-2次） | ⚠️ 少量 | 只在关键剧情说1-2句 |
| 路人/临时角色 | ❌ 禁止 | 街边小贩、围观群众 |

**正确示例**：
```
【旁白】满堂宾客哄笑，指指点点说她是被抛弃的弃妇。
（正确：用旁白描述路人反应，不给路人对话）
```

**错误示例**：
```
宾客甲："听说了吗？王妃被休了！"
宾客乙："啧啧，真是丢人。"
（错误：路人不应该有对话，显得刻意）
```

## 压缩策略

### 内容压缩比例

| 内容类型 | 处理方式 | 压缩比例 |
|----------|----------|----------|
| 心理描写 | 转为对话/表情 | 删除90%+ |
| 环境描写 | 提取关键词给旁白 | 删除95%+ |
| 背景铺垫 | 融入对话自然交代 | 删除80%+ |
| 打斗细节 | 保留关键动作节点 | 删除70%+ |
| 感情戏 | 保留冲突对话 | 删除60%+ |

### 保留原则

**必须保留**：
- 核心冲突和矛盾
- 关键反转点
- 角色特征对话
- 情绪高潮时刻
- 爽点/打脸场景

**可以删除**：
- 日常流水账
- 重复的内心戏
- 过渡性场景
- 非核心支线

**禁止使用**：

- 生理反应：泪水、血液、呕吐
