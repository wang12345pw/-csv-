# AI视频分镜角色信息提取指令（对话式大模型专用）

**版本**: v2.4
**适用**: 即梦AI、可灵AI等视频生成工具
**更新**: 2025-12-11
**输出格式**: Markdown表格（直接输出，无需其他内容）

---

## 一、任务目标

从短剧原文中提取所有角色信息，生成**角色信息表**（Markdown表格格式）：
- 包含每个角色的完整外貌描述（用于导入AI生成角色设计图）
- 包含每个角色的音色描述（用于AI配音）
- **重要**：同一角色如有明显身份/阶段变化，需分别生成不同阶段的外貌描述
- **阶段角色使用"图一引用"机制**，确保人物形象一致
- **同一角色的不同阶段共用相同音色**（音色不随身份变化而改变）

---

## 二、工作流程

```
剧本原文 → 统计角色出现次数 → 筛选角色(≥2次) → 分析角色阶段变化 → 设计基础形象+阶段服装+音色 → 生成Markdown表格
```

**执行规则**：
- 收到剧本后直接开始处理，无需确认
- **只输出Markdown表格，不输出任何其他内容**
- **重点分析角色的身份/阶段变化**，为不同阶段设计不同着装

---

## 三、角色阶段分析规则（重要！）

### 3.1 什么是"阶段变化"

**阶段变化**指角色在剧情中经历的**重大变化**，导致外貌/着装风格明显不同，需要单独生成角色设计图。

**需要区分阶段的情况**：
| 变化类型 | 示例 | 阶段命名 | 说明 |
|---------|------|---------|------|
| 身份晋升 | 丫鬟→贵人→贵妃 | 苏婉、苏婉/贵人、苏婉/贵妃 | 服装档次提升 |
| 身份下降 | 贵妃→废妃→乞丐 | 苏婉、苏婉/废妃、苏婉/落魄 | 服装档次下降 |
| 地位变化 | 乞丐→富商→官员 | 张三、张三/富商、张三/官员 | 服装风格转变 |
| 职位晋升 | 小兵→将军→皇帝 | 李四、李四/将军、李四/皇帝 | 服装档次提升 |
| 年龄跨度 | 6岁→12岁→18岁 | 沈缇、沈缇/12岁、沈缇/成年 | 身高+发型+服装变化 |
| 处境变化（富→穷） | 富家小姐→家道中落 | 苏婉、苏婉/落魄 | 华服→粗布衣 |
| 处境变化（穷→富） | 穷书生→高中状元 | 张三、张三/状元 | 粗布衣→华服 |
| 健康变化 | 健康→病弱/受伤 | 苏婉、苏婉/病中 | 服装+气质变化 |

**不需要区分阶段的情况**：
- 仅换了一件衣服，但身份/处境不变
- 场景切换但身份地位没变（如从卧室到花园）
- 短暂的临时装扮（如乔装打扮一场戏）

### 3.2 阶段命名规范

**格式**：
- 基础角色：`角色名`（第一个出场的身份）
- 阶段角色：`角色名/阶段名`

**常用阶段名**：
| 变化类型 | 常用阶段名 |
|---------|----------|
| 女性身份晋升 | 丫鬟、宫女、贵人、嫔妃、贵妃、皇后、太后 |
| 男性身份晋升 | 小厮、小兵、校尉、将军、王爷、皇帝 |
| 年龄变化 | 6岁、9岁、12岁、童年、少年、成年 |
| 处境下降 | 落魄、流浪、乞丐、废妃、囚犯 |
| 处境上升 | 富贵、巅峰、状元、当家 |
| 健康变化 | 病中、受伤、康复 |

## 四、角色筛选规则

### 4.1 纳入标准
- **凡是有台词（说话）的角色都必须生成角色设计图，旁白除外**
- **出现次数≥2次**的角色必须纳入
- 有台词、有名字、对剧情有影响的角色优先
- 主角、重要配角必须纳入

### 4.2 排除标准
- 仅出现1次的临时角色（路人、群众）
- 无名无姓的背景角色（如"侍卫甲"、"宫女乙"仅提及一次）
- 临时角色在分镜中直接描述外貌，不使用@引用

### 4.3 角色类型分类
| 类型 | 说明 | 示例 |
|------|------|------|
| 主角 | 剧情核心人物 | 女主、男主 |
| 重要配角 | 多次出现、推动剧情 | 皇后、皇子、贵妃 |
| 一般配角 | 出现2-5次 | 贵人、嬷嬷、管事 |
| 功能性角色 | 特定功能、多次出现 | 太监、丫鬟 |

## 六、角色信息表生成规范（Markdown表格格式）

### 6.1 两种角色描述格式

⚠️ **核心机制**：基础角色使用完整描述，阶段角色使用"图一引用"格式

#### 基础角色描述（完整格式）

**适用**：每个角色的第一条记录（序号最小的那条）

**固定前缀**: `角色设计图，正面视角，全身，白色背景，`

**必含要素**（按顺序）:
1. 角色定位（一位古风青年女子/男子/老妇人/男童等）
2. 年龄+身高（如：十八岁，身高约一米六二）
3. 发型详细描述（发型名称+发饰+发色）
4. 脸型+五官（眉型、眼型+眼神、鼻型、唇型）
5. 气质描述（2-4个词）
6. 服装详细描述（颜色+款式+纹饰+配饰）
7. **鞋子描述**（颜色+款式）⚠️ 必须包含
8. 站立姿势

**字数**: 100-180字（含前缀）

**基础角色示例**：
```
角色设计图，正面视角，全身，白色背景，一位古风青年男子，二十岁，身高约一米八〇，乌黑短发简单束起用布带系住，棱角分明剑眉星目，深邃墨眸透着坚毅果敢，高挺鼻梁英气逼人，薄唇紧抿，气质刚毅沉稳，深蓝色粗布短褐简朴结实，腰系布带佩戴水囊，脚穿深蓝色布靴，站立姿势
```

#### 阶段角色描述（图一引用格式）

**适用**：同一角色的后续阶段（如 角色名/将军、角色名/皇帝、角色名/12岁、角色名/落魄）

⚠️ **三种阶段类型**：

**A. 身份变化阶段**（年龄不变，只换服装）
- 适用：丫鬟→贵妃、小兵→将军等身份晋升/下降
- 格式：`图一中穿着[基础服装颜色][服装类型]的[称谓]，穿着[阶段服装详细描述]，站立姿势，白色背景`

**B. 年龄变化阶段**（年龄+身高+服装都变）⚠️ 重要！
- 适用：6岁→12岁→18岁等年龄跨度
- 格式：`图一中的[称谓]从[原年龄]岁长到[新年龄]岁，身高从[原身高]长到[新身高]，[发型变化描述]，穿着[阶段服装详细描述]，站立姿势，白色背景`

**C. 处境变化阶段**（服装档次大变，年龄不变）⚠️ 重要！
- 适用：富→穷（华服→粗布）、穷→富（粗布→华服）、健康→病弱等
- 格式：`图一中穿着[基础服装颜色][服装类型]的[称谓]，[处境变化描述]，穿着[阶段服装详细描述]，站立姿势，白色背景`

**要素说明**：
- `图一中的[称谓]` / `图一中穿着...的[称谓]`：引用基础角色，确保AI识别同一人物
- `从[原年龄]岁长到[新年龄]岁，身高从[原身高]长到[新身高]`：**年龄变化阶段必须包含**
- `[处境变化描述]`：如"家道中落"、"一夜暴富"、"病中消瘦"等
- `[发型变化描述]`：年龄变化时发型也应相应变化（如双辫→低髻→高髻）
- `穿着[阶段服装详细描述]`：新阶段的完整服装描述（颜色+款式+纹饰+配饰+鞋子）
- `站立姿势，白色背景`：固定结尾

**身份变化示例**（身份晋升，年龄不变）：
```
图一中穿着深蓝色短褐的男子，穿着银色锦缎战袍绣金色云纹外罩精致铠甲威武英气，腰系金色革带佩戴宝剑玉佩，脚穿银色战靴，站立姿势，白色背景
```

## 九、特殊情况处理

### 9.1 角色阶段判断
**需要分阶段**：身份/地位有重大变化，着装风格明显不同
**不需要分阶段**：仅场景切换，身份地位不变

### 9.2 角色数量过多（>20条记录）
优先保留：主角的所有阶段 > 重要配角 > 一般配角
控制总记录数在20条以内

### 9.3 双女主/双男主
使用不同色系区分：
- 女主A：淡粉色系（各阶段保持粉色系引用）
- 女主B：玫红色系
- 男主A：深蓝色系
- 男主B：深红色系

### 9.4 反派角色
使用深色、暗色系：藏青色、暗红色、深紫色、黑色
